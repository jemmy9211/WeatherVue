# WeatherVue  
### Introduction  
此網頁串接中華民國(台灣)氣象局的公開API，利用該API提供的資料進行視覺化處理，使複雜的資料可以讓使用者讀取容易，此專案使用  
到的技術有Vue、Bootstrap、leaflet，上述兩樣工具使網頁的搭建快速且實用，以下整理該網頁主要亮點  


1. 提供全台觀測站即時資訊(溫度、風速、預報...)  
2. 提供觀測站的位置定位
3. 全台觀測站地圖(點按地圖mark可得到待觀測站資訊)  
4. 可搜尋觀測站名稱或是地名來取得相關觀測站的資料
